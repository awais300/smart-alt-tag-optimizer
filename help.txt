Post-Installation Steps
1. Install Composer Dependencies
bashcd smart-alt-tag-optimizer
composer install

2. Verify Structure
bash# Should output all PHP files
find src -name "*.php" | head -20

3. Activate in WordPress
bash# Symlink or copy to wp-content/plugins
ln -s /path/to/smart-alt-tag-optimizer wp-content/plugins/

# Or copy directly
cp -r smart-alt-tag-optimizer wp-content/plugins/

4. Run Tests (Optional)
bashcomposer test
